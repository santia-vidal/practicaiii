Los tres pilares de la Web

Tim_Berners-Lee es considerado el padre de la Web porque desarrolló los tres elementos históricos para el funcionamiento de la Web.

  • HTML
  • HTTP
  • URL

HTML

HTML, siglas de HyperText Markup Language , hace referencia al lenguaje de marcado para la elaboracion de paginas web. Es un estándar que sirve de referencia para la elaboración de páginas wen en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes, entre otros. Es un estándar a cargo de la E#C, organización dedicada a la estandarización de casi todas las tecnologías ligadas a la web, sobre todo en lo referente a su escritura e interpretación. Es un lenguaje con el que se definen las páginas web.
